РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
«Модель кассового терминала супермаркета»

1. ЗАПУСК ПРОГРАММЫ
   - Откройте решение CashRegister.sln в Visual Studio 2022 или новее.
   - Убедитесь, что установлен .NET SDK 10.0.300.
   - Соберите и запустите проект CashRegister (F5).
   - На титульной форме нажмите «Перейти к работе».

2. АВТОРИЗАЦИЯ КАССИРА
   - Вкладка «Авторизация»: выберите кассира, введите пароль.
   - Тестовые данные:
       Карта 1001, пароль 1111 — Иванова А.С.
       Карта 1002, пароль 2222 — Петров И.И.
       Карта 1003, пароль 3333 — Сидорова М.К.
   - После входа доступна вкладка «Касса».

3. РАБОТА С ЧЕКОМ
   - «Новый чек» — очистка таблицы (при выручке ≥ 200 000 руб. сначала сдайте выручку).
   - Выберите товар в списке (ComboBox) или введите штрих-код вручную.
   - Штучный товар: укажите количество (NumericUpDown).
   - Весовой товар: введите вес в кг (до 3 знаков после запятой).
   - «Добавить в чек», «Удалить последнюю» — управление позициями.

4. ЗАКРЫТИЕ ЧЕКА
   - Введите сумму от покупателя.
   - «Закрыть чек» — расчёт сдачи, печать в окне справа.
   - Если в чеке алкоголь и время ≥ 22:00 — продажа запрещена.
   - Если алкоголь и время < 22:00 — подтвердите возраст покупателя.

5. ВЫРУЧКА
   - При накоплении 200 000 руб. новый чек блокируется.
   - «Сдать выручку» — подтвердите остаток 5 000 руб. в кассе.

6. БАЗА ДАННЫХ
   - Файл supermarket.db создаётся автоматически при первом запуске
     в каталоге с исполняемым файлом (bin\Debug\net10.0-windows\).
   - Таблицы: Products, Cashiers, AppSettings.

7. СОСТАВ ПРОЕКТА
   TitulForm.cs      — титульная форма
   MainForm.cs       — основная форма кассы
   HelpForm.cs       — справка
   AgeVerificationForm.cs — проверка возраста
   RevenueForm.cs    — сдача выручки
   Models.cs         — модели данных
   DatabaseHelper.cs — работа с SQLite
   CashRegisterLogic.cs — бизнес-логика
